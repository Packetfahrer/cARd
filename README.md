# cARd

Simple demo of animated card made with ARKit + SceneKit. Demo video is here: https://twitter.com/RosberryApps/status/894418114426941444

## Requirements

- iOS 11.0+
- Xcode 9.0+

## Installation

Clone the repo and just run.

## Usage

- Run on your device
- Find flat surface
- Tap on it
- Have fun!
